The the web API to interact with this CorDapp, end-points available:

  • GET /api/tdbo/tdbos (Muestra TDBOs)
  • GET /api/tdbo/emitir-tdbo?cantidad=INTEGER&moneda=[GBP/USD]&participante=NODE_NAME (emitir un TDBO a un participante con una cantidad y moneda especifica)
  • GET /api/tdbo/transferir-tdbo?id=LINEAR_ID&participante=NODE_NAME (Transferir un TDBO especificado por linearId a otro participante
  • GET /api/tdbo/liquidar-tdbo?id=LINEAR_ID&cantidad=INTEGER&moneda=[GBP/USD] (liquidar una cantidad de un TDBO especificado por linearId)
  • GET /api/tdbo/cash (muestra todos los cash states)
  • GET /api/tdbo/auto-emitir-cash?cantidad=INTEGER&moneda=[GBP/USD] (auto emite una cantidad de cash en la moneda especificada)
  • GET /api/tdbo/cash-balances (muestra los balances de cash por moneda)
